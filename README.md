## 二、部署步骤（3步即可上线）
### 方式1：部署到GitHub Pages
1. 新建GitHub仓库，命名：`linghuaqian-assistant`
2. 上传所有文件到仓库（确保index.html在根目录）
3. 仓库设置 → Pages → 选择main分支 → 保存，等待5分钟即可访问。

### 方式2：部署到服务器/虚拟主机
1. 把整个文件夹上传到服务器根目录（如：/wwwroot/）
2. 配置域名指向该文件夹
3. 直接访问域名即可使用（无需安装/编译/数据库）

### 方式3：部署到Netlify/Vercel（免费）
1. 注册Netlify/Vercel账号
2. 导入GitHub仓库，自动部署（无需手动配置）

## 六、常见问题
Q：对话记录会丢失吗？
A：不会，对话记录保存在浏览器本地存储（localStorage），清空浏览器缓存才会丢失。

Q：需要后端/数据库吗？
A：不需要，纯前端实现，直接部署即可用。

Q：支持多用户吗？
A：当前版本为单用户本地版，如需多用户可扩展后端接口。
